<h1>🌤️ SkyCast Weather App</h1>

<li>A lightweight, responsive weather application built with vanilla JavaScript that provides real-time weather data using the OpenWeatherMap API.</li>

## 🚀 Features
- **Live Search:** Get instant weather updates for any city in the world.
- **Detailed Metrics:** Displays temperature, humidity, wind speed, and weather descriptions.
- **Dynamic Styling:** The UI updates visually based on the current weather conditions.
- **Fully Responsive:** Works seamlessly on mobile, tablet, and desktop devices.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling with Flexbox and glassmorphism effects.
- **JavaScript (ES6):** Asynchronous API calls (Fetch API) and DOM manipulation.
- **API:** [OpenWeatherMap API](https://openweathermap.org/api).
